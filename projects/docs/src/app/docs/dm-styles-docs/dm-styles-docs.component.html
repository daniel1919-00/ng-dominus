Lightweight library mainly build to style the dominus components demo page, but can be used on any site along any other css libraries.

The library takes a utility-first approach containing only structural syles and does not require a theme.

All classes are prefixed with the dm:* which is used as a namespace. As such, the library can be used alongside any other css frameworks/libraries as long as this namespace is unique.

Some classes have a "forced" version, suffixed with *-f which makes their properties !important.
